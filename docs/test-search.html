테스트 시나리오:
  1. 문서 로드 (다양한 텍스트 포함)
  2. Ctrl+F로 찾기 다이얼로그 열기
  3. 텍스트 검색 및 하이라이트
  4. 다음/이전 찾기 (F3/Shift+F3 또는 다이얼로그 버튼)
  5. Ctrl+H로 바꾸기 모드 전환
  6. 개별 교체 및 모두 교체
  7. Undo/Redo 테스트

⌨️ 키보드 단축키

  • Ctrl+F (Mac: Cmd+F) - 찾기 다이얼로그
  • Ctrl+H (Mac: Cmd+H) - 바꾸기 다이얼로그
  • Enter - 다음 찾기
  • Shift+Enter - 이전 찾기
  • ESC - 다이얼로그 닫기
  • Ctrl+Z - 실행 취소
  • Ctrl+Y - 다시 실행
