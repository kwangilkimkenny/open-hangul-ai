📋 Clipboard Command Test

테스트 시나리오:
  1. 문서 로드 및 커서 표시
  2. 텍스트 선택 (드래그 또는 Shift+화살표)
  3. Copy (Ctrl+C 또는 버튼)
  4. Cut (Ctrl+X 또는 버튼)
  5. Paste (Ctrl+V 또는 버튼)
  6. Undo/Redo 테스트

⌨️ 키보드 단축키

  • Ctrl+C (Mac: Cmd+C) - 복사
  • Ctrl+X (Mac: Cmd+X) - 잘라내기
  • Ctrl+V (Mac: Cmd+V) - 붙여넣기
  • Ctrl+Z (Mac: Cmd+Z) - 실행 취소
  • Ctrl+Y (Mac: Cmd+Y) - 다시 실행
  • Shift+화살표 - 텍스트 선택 (키보드)
  • 마우스 드래그 - 텍스트 선택
