핵심 기능

HWPX 네이티브 지원

브라우저에서 직접 HWPX 파일 처리. 서버 변환 불필요.

  • 완전한 OWPML 파싱
  • HWP 자동 변환
  • 다중 포맷 지원

AEGIS 보안 시스템

AI 입출력 보안 및 개인정보 자동 보호.

  • 219개 공격 패턴 차단
  • PII 자동 가명처리
  • 한국어 특화 방어

TruthAnchor 검증

AI 출력 신뢰도 검증 및 할루시네이션 방지.

  • 6개 도메인 가드레일
  • 수치 정확도 검증
  • 실시간 사실 확인

설치

기본 사용

import { HWPXViewer } from 'open-hangul-ai';

function App() {
  return (
    <HWPXViewer
      fileUrl="/문서.hwpx"
      enableAI={true}
    />
  );
}
