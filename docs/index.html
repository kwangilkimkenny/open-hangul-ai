Our Philosophy

닫힌 편집기의 시대를,
오픈소스로 끝냅니다

한글 문서는 공공·기업·개인의 가장 중요한 기록입니다. 그 편집기가 특정 벤더·특정 OS·특정 포맷에 묶여 있을 수는 없습니다. 오픈한글AI는 투명한 코드와 열린 생태계로 한글 문서의 미래를 다시 설계합니다.

01

완전한 오픈소스

한 줄의 코드까지 GitHub에 공개되어 있습니다.

MIT 라이선스로 누구나 읽고, 고치고, 자유롭게 배포할 수 있습니다. 블랙박스 없는 투명한 문서 기술 — 한글의 미래는 한 회사가 아닌 커뮤니티가 함께 만듭니다.

  • MIT License · 상업적 사용 제한 없음
  • 코드 · 이슈 · 로드맵 모두 공개
  • Pull Request · 기여자 언제든 환영
02

기존 한글 편집기의 한계를 넘어

플랫폼 종속, 무거운 설치, 사라지는 호환성 — 끝냅니다.

수십 년간 반복된 한글 편집기의 고질적 문제를 웹 표준과 브라우저 네이티브 처리로 근본부터 다시 풉니다. 서버 변환도, 외부 SaaS도 필요 없습니다.

  • 설치 없이 브라우저에서 바로 열기
  • OWPML 표준 기반 장기 보존성
  • OS · 디바이스 · 플랫폼 독립
03

사람과 AI, 모두에게 친화적

사람은 더 빠르게, AI는 더 정확하게 — 같은 문서를 함께.

사람에게는 직관적인 UI와 안전한 개인정보 보호를, AI에게는 구조화된 컨텍스트와 검증 가능한 접근을. Human-friendly × AI-friendly를 동시에 설계한 첫 한글 편집기입니다.

  • Human : 직관적 UX · WCAG 접근성
  • AI : 구조화 파싱 · 컨텍스트 API · 출처 추적
  • 양측이 안전하게 협업하는 단일 문서 레이어

Core Features

다른 솔루션이 못하는 것을 합니다

3가지 차별화 기능과 3가지 개발자 편의 기능으로, 한글 문서 AI의 새로운 표준을 만듭니다.

HWPX 네이티브 지원

브라우저에서 직접 HWPX/HWP 파일을 처리합니다. 서버 변환·외부 SaaS 의존성이 0.

  • 완전한 OWPML 파싱
  • HWP → HWPX 자동 변환
  • PDF · DOCX · TXT 통합 지원

AEGIS 보안 시스템

AI 입출력을 실시간 스캔. 한국어 특화 219개 공격 패턴을 사전 차단하고 PII를 자동 가명처리합니다.

  • 219개 프롬프트 인젝션 패턴
  • 주민번호 · 카드 · 연락처 자동 마스킹
  • 한국어 특화 우회 공격 방어

TruthAnchor 검증

AI 출력의 수치·인용·사실관계를 75개 규칙으로 교차 검증해 할루시네이션을 차단합니다.

  • 6개 도메인별 가드레일
  • 수치·날짜·인용 정확도 검증
  • 출처 추적 가능한 응답

규제 준수 자동 평가

4개 글로벌 AI 규제 프레임워크 기준으로 자동 점검합니다.

  • 한국 AI 기본법 · EU AI Act
  • NIST AI RMF · ISO/IEC 42001

AI 문서 생성

200만 토큰 컨텍스트로 대용량 참조 문서 기반 초안을 스트리밍 생성합니다.

  • 다중 참조 · 인용 추적
  • 실시간 스트리밍 응답

개발자 경험

5분 안에 설치 끝. 완전한 타입 지원과 964개 테스트로 검증된 안정성.

Getting Started

5분 안에 시작하기

npm 설치부터 보안·검증 활성화까지, 3 단계면 충분합니다.

1

설치

# 패키지 설치
npm install open-hangul-ai
2

기본 사용

import { HWPXViewer } from 'open-hangul-ai';

function App() {
  return (
    <HWPXViewer
      fileUrl="/문서.hwpx"
      enableAI={true}
    />
  );
}
3

보안 · 검증 활성화

const config = {
  security: { enableAEGIS: true },
  verification: { enableTruthAnchor: true },
  compliance: ['K-AI-Act', 'EU-AI-Act']
};
