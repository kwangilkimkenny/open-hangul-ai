✍️ Text Formatting Command Test

테스트 시나리오:
  1. 문서 로드 (다양한 텍스트 포함)
  2. 텍스트 선택
  3. 포맷 버튼 클릭 또는 키보드 단축키 사용
  4. 포맷 변경 확인 (Bold, Italic, Underline, Strikethrough)
  5. 여러 포맷 조합 테스트
  6. Undo/Redo 테스트

⌨️ 키보드 단축키

  • Ctrl+B (Mac: Cmd+B) - Bold (굵게)
  • Ctrl+I (Mac: Cmd+I) - Italic (기울임)
  • Ctrl+U (Mac: Cmd+U) - Underline (밑줄)
  • Ctrl+Shift+X (Mac: Cmd+Shift+X) - Strikethrough (취소선)
  • Ctrl+Z - 실행 취소
  • Ctrl+Y - 다시 실행
