# 09 · Numbering & Bullets (번호매기기·글머리표)

## 검증 의도

- 다단계 번호 매기기 (1, 1.1, 1.1.1 …)
- 한국어 번호 매기기 (가나다, ㄱㄴㄷ, 일이삼)
- 글머리 기호 (•, ○, ■, 사용자 문자)
- 번호 시작 값·접두/접미사

## 기대 결과

- 파서: `numbering` 정의 + 각 문단의 level / index
- 렌더러: 들여쓰기와 함께 정확한 번호 표시
- Round-trip: 재시작 / 이어 매기기 유지

## 회귀 시나리오

1. 한 단계 1, 2, 3 …
2. 두 단계 (1, 1.1)
3. 한국어 "가, 나, 다"
