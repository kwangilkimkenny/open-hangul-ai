# 05 · Footnote / Endnote (각주·미주)

## 검증 의도

- 본문에서 각주 참조 마커 (번호 / 사용자 기호)
- 각주 내용 문단
- 자동 번호 / 페이지 단위 재시작
- 미주 (endnote) 구분

## 기대 결과

- 파서: 본문 문단의 inline `footnote` 노드 + 각주 본문 페어 추출
- 렌더러: 본문 끝(또는 페이지 하단)에 각주 번호와 함께 표시
- Round-trip: 번호 자동 재배치 후에도 참조 무결

## 회귀 시나리오

1. 한 문단에 각주 1개
2. 한 페이지에 각주 2개 (번호 1, 2)
3. 미주 (문서 끝에 모임)
