# 06 · Header / Footer (머리말·꼬리말)

## 검증 의도

- 모든 페이지 / 홀수 페이지 / 짝수 페이지 / 첫 페이지 분리
- 안에 페이지 번호 필드, 표, 이미지 가능
- 섹션별로 다른 머리말/꼬리말

## 기대 결과

- 파서: `headerFooter` 노드 + applyTo (all/odd/even/first) 보존
- 렌더러: 짝홀에 다른 머리말 출력
- Round-trip: 섹션 분리 시 머리말 매핑 유지

## 회귀 시나리오

1. 모든 페이지 머리말 "문서제목"
2. 홀수 페이지 꼬리말 "쪽 N", 짝수 페이지 꼬리말 "N 쪽"
3. 첫 페이지 꼬리말 없음
