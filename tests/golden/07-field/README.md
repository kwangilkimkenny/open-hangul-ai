# 07 · Field (필드)

## 검증 의도

- 날짜 (`DATE`), 시간 (`TIME`)
- 페이지 번호 (`PAGE_NUM`, `TOTAL_PAGE`)
- 파일 경로 / 사용자 정의 (`USER_DEFINED`)
- 필드 시작 ↔ 종료 마커 짝 유지

## 기대 결과

- 파서: 필드 종류 + 옵션 + 표시 텍스트 분리
- 렌더러: 사용자 정의 필드의 fallback 텍스트 표시
- Round-trip: 필드 마커 짝 유지

## 회귀 시나리오

1. 본문 안 페이지 번호 필드
2. 머리말의 날짜 필드 (YYYY-MM-DD 포맷)
3. 사용자정의 필드 + 기본값 fallback
